ЗАКАЧКА ФАЙЛА

 Закачивать файл на сервер требуется для того, чтобы его могли скачивать посетители Вашего сайта. Чтобы посетители могли скачать файл с Вашего сайта необходимо расположить ссылку (как это сделать смотрите в разделе "Редактирование страниц") на этот файл в определенной странице.
 Чтобы закачать файл кликните в верхней части административного раздела кликните по ссылке "Закачать файл". Затем нажмите на кнопку "Обзор", откроится окно для выбора файла на Вашем компьютере. Найдите файл, который необходимо закачать и нажмите кнопку "Открыть". Теперь нажмите кнопку "Закачать". После этого указанный файл закачается на сервер.
К оглавлению.

 
ЗАКАЧКА ИЗОБРАЖЕНИЙ

 Закачать файл изображения на сервер нужно для того, чтобы это изображение можно было вставлять в дальнейшем на страницы (как вставлять изображения в страницы смотрите в разделе "Редактирование страниц").
 Чтобы браузеры посетителей могли отобразить Ваши изображения, закачиваемые изображения должны быть в одном из трех форматов: *.jpg *.gif *.png.
 Чтобы закачать изображение кликните в верхней части административного раздела кликните по ссылке "Закачать изображение". Затем нажмите на кнопку "Обзор", откроится окно для выбора файла на Вашем компьютере. Найдите файл, который необходимо закачать и нажмите кнопку "Открыть". Теперь нажмите кнопку "Закачать". После этого указанное изображение закачается на сервер.
К оглавлению.

РЕДАКТИРОВАНИЕ СТРАНИЦ

Визуальный редактор страниц позволяет изменить внешний вид редактируемого раздела, добавить текст, изображения, ссылки и прочие элементы вэб-страниц. Перечислим в порядке представления управляющих кнопок вверху редактора их назначения.
Сохранение текущей страницы в качетсве самостоятельной html-страницы на Вашем компьютере. При нажатии на эту иконку появится окно, в котором надо определить месторасположение сохраняемого файла и его имя. Задайте требуемые параметры и нажмите кнопку "Сохранить". Файл будет сохранен в указанной Ваи папке.
Печать текущей старницы. При нажатии на этой иконке появляется стандартный запрос печати. Если нажать кнопку "Печать", то на принтер выводится текст текущей страницы.
Вырезание выделенного текста или фрагмента документа в буфер обмена. При вырезке выделнный фрагмент удаляется и сохраняется только в буфере обмена. Этот фрагмент в последующем можно вставить в другие документы для дальнейшего редактирования.
Копирование выделенного текста или фрагмента документа. При копировании выделенного фрагмента, его содержимое помещается в буфер обмена. Эту информацию в дальнейшем можно вставить в другой документ на Вашем компьютере или в другую редактируемую страницу сайта.
Вставить фрагмент из буфера обмена. Если в Вашем буфере обмена уже есть какая-то информация, то при клике на эту иконку информация будет вставлена по месту расположения текущего курсора.
Просмотр страницы в браузере. При клике по этой иконке открывается новое окно браузера, в котором отображается последняя версия текущей страницы.
Отменить. При клике по этой иконке происходит отмена последнего действия (аналогичное действие производит нажатие двух клавиш - Ctrl и Z).
Повторить. При клике по этой иконке происходит повтор последнего действия.
Вставить гиперссылку. При клике по этой иконке появляется окшоко с формой. В ней надо ввести тип ссылки (по умолчанию - это http, этот тип годится для ссылок на внутренние страницы и внешние сайты), URL ссылки и тип открытия ( в новом окне или в том же).
Новый абзац. При клике по этой иконке текущий курсор перемещается на новую строку с пропуском в одну строку.
Новая срока. При клике по этой иконке текущий курсор перемещается на новую строку без пропуска строки.
Горизонтальная линия. При клике по этой иконке сразу после текущего курсора на следующей строке вставляется горизонтальная линия во всю ширину страницы.
Вставить изображение. При клике по этой иконке появляется окно со списком изображений. Выберите необходимое изображение и нажмите кнопку "Вставить". После этого редактор поместит изображение в место текущего курсора. Если файл изображения еще не закачан на сервер, то нажмите на кнопку "Закачать файл" и следуйте инструкциям.
Бегущая строка. При клике по этой иконке по месту текущего курсора появляется рамка с маркерами. В ней необходимо ввести текст. После ввода текста и публикации страницы на сайте в указанном месте появится бегущая строка с Вашим текстом.
Вставка таблицы. Создание новой таблицы может быть необохдимым, например, для форматирования текста в две колонки. При клике по этой иконке открывется новое окно с параметрами будущей таблицы. В поле "Ширина таблицы" укажите общуюю ширину таблицы в процентах от ширины блока куда она вставляется или в пикселях (в таком случае ширина таблицы не будет изменятся при изменении ширины окна барузера). В поле "Количество строк" введите число равное количеству строк будущей таблицы. То же касается и количества столбцов. Поле "Ширина бордюра" определяет ширину каемки вокруг каждой ячейки таблицы. Введите в это поле 0 или другое число, оно будет равно ширине бордюра в пикселях. Рекомендуется ставить 0 или 1, если вообще хотите обозначить границы таблицы. Поле "Cellpadding" определяет в пикселях отступ внутри каждой ячейки, а поле "Cellspacing" определяет расстояние в пикселях между ячейками. "Цвет фона" - это код цвета внутри всей таблицы. По умолчанию он белый, если хотите, чтобы цвет был таким же как и на странице куда вставляется таблица, то сделайте это поле пустым. Когда все параметры будут установлены остается только нажать на кнопку "Вставить", и таблица будет помещена в место активного курсора.
 Вставить произвольный html-код или текст. Эта команда необходима, если Вы хотите вставить, например, код нового счетчика или баннера в свою страницу. Сначала поместите курсор в то место, где Вы хотели бы видеть счетчик или баннер. Затем кликните по этой иконке и вставьте в появившееся окно код.Нажмите кнопку "Ок", и получите результат.
 Упарвление отображением текста. Эти иконки, а так же выпадающие списки "Шрифт" и "Размер" предназначены для управления внешним видом текста на Вашей странице. Для того, что бы задать шрифт текста необходимо выделить нужный фрагмент текста, а затем в выпадающем тексте выбрать шрифт, и он тут же изменится. Точно так же Вы можете поменять и размер шрифта. Так же, предварительно выделив нужный фрагмент текста, Вы можете его сделать жирным, кликнув по иконке В, наклонным, кликнув по иконке I, подчеркнутым, кликнув по иконке U и перечеркнутым, кликнув по иконке S. Последняя иконка задает цвет текста.
 Управление абзацем. Эта группа иконок задает выравнивание абзаца или выделенного фрагмента по левому краю, по центру или по правому краю соответсвенно.
 Управление списками. Эти две иконки определяют внешний вид списков. Если Вы создете список каких-нибудь наименований, например, то первая иконка вставляет перед каждым пунктом списка черный кружок (маркер), а вторая вставляет номер пункта. Чтобы применить эти функции установить курсор там, где требуются маркеры списка и кликните по соответствующему маркеру. У начала строки вставится маркер, и каждый раз, когда Вы будете нажимать "Enter", будет вставлятся новый маркер.
 Управление отступом абзаца. Эти две иконки позволяют установить отступ абзаца от левого края. Первая иконка уменьшает отступ, а вторая увеличивает. Установите курсор на начало абзаца и кликните по по соответствующей иконке.
Кнопка "Опубликовать". Когда закончите все операции с редактируемой страницей и убедитесь, что сделано все, что Вы хотели нажмите на эту кнопку и измененная страница опубликуется на сайте.

Совет!

  • Для копирования текста в окно редактора рекомендуем использовать не правую кнопку мыши, а сочетание клавиш Ctrl+C для копирования и Ctrl+V для вставки.
  • Избегайте копирования текстов из Ворда, т.к результаты могут быть не предсказуемыми. Лучше пользоваться Блокнотом.
  • Не забывайте перед вставкой изображения в страницу сначала закачать ее на сервер.
  • Для вставки ссылок используйте метод копирования из адресной строки браузера. Просто перейдите на страницу или сайт, на который Вы хотите поставить ссылку, скопируйте текст ссылки из адресной строки и вставьте в окне создания гиперссылки.
  • Не злоупотребляйте разными размерами, цветами и прочими вариантами отображения шрифта. Это приводит к нарушению общего стиля отображения на Вашем сайте. Лучше вообще не задавайте эти параметры. Шаблон Вашего сайта сделает это за Вас.
  • Регулярно заглядывайте в это руководство, пока не будете уверенны, что знаете ВСЕ.

    К оглавлению.
